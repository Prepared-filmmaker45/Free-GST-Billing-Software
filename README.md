## 📋 About Free-GST-Billing-Software

Free-GST-Billing-Software is a free and open-source billing tool designed for Indian GST compliance. It helps you generate tax invoices, proforma invoices, credit notes, and bills of supply. You can manage CGST, SGST, and IGST calculations accurately. The software supports multiple currencies and works completely offline, so you don’t need an internet connection or sign up to use it. This software suits small businesses, freelancers, and anyone who wants an easy way to create GST-compliant invoices. It runs on Windows without extra software or coding skills.

## 🧭 How to Use the Software

This section explains the basic steps to create your first GST invoice.

### Creating a New Invoice

1. Open the software.
2. Click “New Invoice” or “Create Invoice.”
3. Fill in customer details:
   - Name
   - Address
   - GST number (if applicable)
4. Add items or services:
   - Enter description
   - Enter quantity and rate
   - Select applicable tax rates (CGST, SGST, IGST) will calculate automatically.
5. Choose invoicing type:
   - Tax Invoice
   - Proforma Invoice
   - Credit Note
   - Bill of Supply
6. Add any additional notes or terms.
7. Save the invoice.

---

### Generating PDF and Printing

Once the invoice is complete:

- Click “Generate PDF” to save a copy on your computer.
- Use the “Print” option to get a physical copy.

The software formats invoices to match Indian GST rules and includes HSN codes automatically where required.

---

### Managing Multiple Currencies

- To bill in other currencies, select the required currency from the drop-down menu.
- All calculations adjust accordingly.
- You can switch currencies anytime while creating an invoice.
